# R32 Basketball Stats Ver.3.1.1

## 更新内容
- スマホ版の表示構成を指定UIに合わせて調整
- 下部タブの「スタッツ」を「分析」に変更
- ホーム／試合詳細／分析画面の個人スタッツ表示を同じ構成に統一
- POINTS・AST・BLK・STL・REB・TOを紫アクセントで統一
- 2P成功・3P成功・FT成功は白文字に統一
- 試合一覧タップで試合詳細＋個人スタッツ表示
- 個人スタッツ修正時に登録済み数値を保持したままフォームを表示
- 個人スタッツ削除ボタンを追加
