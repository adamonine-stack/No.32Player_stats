# R32 Basketball Stats Ver.2.9

## 更新内容
- スマホ版ホーム上部のR32壁紙イメージを復元
- 「成長を、数字で見える化。」の見出しを削除し、画像を目立たせる構成へ調整
- PC版左サイドバーにバッシュ＋ボールのイメージを配置
- スマホ版試合登録フォームの各マス幅を揃え、日付とQ数が干渉しないよう調整
- スタッツ画面の選手選択・対象選択のサイズを統一
- 対象データがない場合に「対象データなし」と表示
- 試合別スタッツの対象選択を試合ID基準に修正し、登録済みスタッツが反映されるよう改善
- STL・REB・TOの合計数を白表示へ変更
- 試合一覧を新しい日付順で縦表示
- 試合をタップすると「スタッツ登録」「修正」「削除」を表示
- スタッツ登録は、試合選択済みの状態で選手選択から入力開始
- 試合削除時に関連スタッツも削除し、画面へ即時反映

## アップロード方法
ZIPを解凍し、中身をGitHubリポジトリ直下へ上書きアップロードしてください。
